## 1. Falta o critério de falsidade

A tese diz que cada observação é uma verdade relativa honesta. Mas você mesmo abriu uma exceção: "o que tem intenção de ser falso" não é verdade. Isso significa que existe falsidade — logo, nem toda observação é verdade relativa. Algumas são distorções deliberadas.

Então a tese precisa de um critério que separe **observação parcial honesta** de **distorção**. Sem ele, qualquer mentira reivindica o status de "verdade relativa" e você não tem como recusar.

Este é o coração filosófico do doutorado. Não é detalhe. É o trabalho.
